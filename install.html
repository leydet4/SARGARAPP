Install the App

This site can be installed like an app (PWA) so it shows up as an icon on your phone/tablet and opens full screen.

Loading install options...

If you don’t see an install button, use the steps below for your device.

iPhone / iPad (Safari)

  1. Open this site in Safari (not Chrome).
  2. Tap Share (square with arrow up).
  3. Tap Add to Home Screen.
  4. Name it SAR Forum and tap Add.

Android (Chrome)

  1. Open this site in Chrome.
  2. Tap the ⋮ menu.
  3. Tap Install app or Add to Home screen.
  4. Confirm install.
